Este projeto foi desenvolvido como parte das atividades acadêmicas do curso da Cruzeiro do Sul, com o objetivo de criar um site responsivo e funcional utilizando HTML e CSS.
O site contém três páginas principais; Início, projeto e cadastro e foi estruturado para demonstrar conceitos de design responsivo, hierarquia de conteúdo e navegação intuitiva.

Tecnologias Utilizadas:
HTML5 → Estruturação semântica do conteúdo
CSS3 → Estilização e layout responsivo
Flexbox / Grid Layout → Organização dos elementos visuais
Design Responsivo → Compatibilidade com dispositivos móveis
